Tricep Workout

Workout Information

1) Double Arm Overhead Tricep Extension

  1. Sets: 3
  2. Reps: 10-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Triceps, shoulders

2) Single Arm Overhead Tricep Extension

  1. Sets: 3
  2. Reps: 10-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Triceps, shoulders

3) Diamond Push-Ups

  1. Sets: 3
  2. Reps: 8-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Triceps, chest

4) Dips

  1. Sets: 3
  2. Reps: 8-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Triceps, chest

5) Cable Tricep Pushdown

  1. Sets: 3
  2. Reps: 10-15
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Triceps
