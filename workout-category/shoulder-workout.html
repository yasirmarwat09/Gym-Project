Shoulder Workout

Workout Information

1) Scaption Raises

  1. Sets: 3
  2. Reps: 10-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Shoulders, upper back

2) Handstand Push-Ups

  1. Sets: 3
  2. Reps: 5-10
  3. Rest: 90 seconds between sets
  4. Primary muscles targeted: Shoulders, upper chest

3) Dumbbell Shoulder Press

  1. Sets: 3
  2. Reps: 8-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Shoulders, triceps

4) Upright Row

  1. Sets: 3
  2. Reps: 10-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Shoulders, traps

5) Front Raises

  1. Sets: 3
  2. Reps: 10-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Front deltoids

6) Lateral Raises

  1. Sets: 3
  2. Reps: 10-15
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Lateral deltoids

7) Overhead Press

  1. Sets: 3
  2. Reps: 8-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Shoulders, triceps

8) Dumbbell Shrugs

  1. Sets: 3
  2. Reps: 10-15
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Traps, shoulders
