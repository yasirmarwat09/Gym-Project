Legs Workout

Workout Information

1) Deadlifts

  1. Sets: 3
  2. Reps: 8-10
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Hamstrings, glutes, lower back

2) Seated Hip Abduction

  1. Sets: 3
  2. Reps: 10-15
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Hip abductors, glutes

3) Leg Curls

  1. Sets: 3
  2. Reps: 10-12
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Hamstrings

4) Bulgarian Split Squats

  1. Sets: 3
  2. Reps: 8-10 per leg
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Quadriceps, glutes, hamstrings

5) Calf Raises

  1. Sets: 3
  2. Reps: 12-15
  3. Rest: 30-45 seconds between sets
  4. Primary muscles targeted: Calves

6) Leg Press

  1. Sets: 3
  2. Reps: 8-12
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Quadriceps, glutes, hamstrings

7) Lunges

  1. Sets: 3
  2. Reps: 8-12 per leg
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Quadriceps, glutes, hamstrings

8) Squats

  1. Sets: 3
  2. Reps: 8-12
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Quadriceps, glutes, hamstrings, core
