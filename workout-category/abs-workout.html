Abs Workout

Workout Information

1) Plank

  1. Sets: 3
  2. Duration: 30-60 seconds
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Core, shoulders

2) Russian Twist

  1. Sets: 3
  2. Reps: 10-15 per side
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Core, obliques

3) Bicycle Crunches

  1. Sets: 3
  2. Reps: 10-15 per side
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Core, hip flexors

4) Leg Raises

  1. Sets: 3
  2. Reps: 10-15
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Lower abs

5) Toe Touches

  1. Sets: 3
  2. Reps: 10-15
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Upper abs
