Chest Workout

Workout Information

1) Bench Press

  1. Sets: 4
  2. Reps: 8-12
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Middle chest, shoulders, triceps

2) Inclined Bench Press

  1. Sets: 3
  2. Reps: 8-10
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Upper chest, shoulders, triceps

3) Smith Machine Inclined Press

  1. Sets: 3
  2. Reps: 8-10
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Upper chest, shoulders, triceps

4) Chest Press

  1. Sets: 3
  2. Reps: 8-10
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Middle chest, triceps, shoulders

5) Dumbbell Press

  1. Sets: 3
  2. Reps: 8-10
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Chest, shoulders, triceps

6) Dips

  1. Sets: 3
  2. Reps: 8-12
  3. Rest: 60-90 seconds between sets
  4. Primary muscles targeted: Lower chest, triceps, shoulders

7) Push-Ups

  1. Sets: 3
  2. Reps: 10-15
  3. Rest: 60 seconds between sets
  4. Primary muscles targeted: Chest, triceps, shoulders

8) Pec Fly
